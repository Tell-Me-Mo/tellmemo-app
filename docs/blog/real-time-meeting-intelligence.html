The Technical Architecture (For the Curious)

Real-Time Processing Pipeline

  1. Audio Capture: Browser captures microphone audio (PCM 16kHz, 16-bit, mono)
  2. WebSocket Streaming: Audio chunks sent to backend every 100-200ms
  3. AssemblyAI Transcription: Real-time speech-to-text with speaker diarization
  4. GPT-5-mini Streaming: Analyzes transcript as it arrives, detects questions and actions
  5. Answer Discovery: Four parallel tiers run simultaneously
  6. Client Updates: Progressive results streamed back to UI
